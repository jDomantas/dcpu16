Keyboard for DCPU-16

Hardware ID:		0x4B455953
Hardware version:	1
Manufacturer ID:	0x44464C54

Interrupts:

A = 0:
	Removes character from input buffer	and writes
	it to register C, or 0 if buffer is empty.
A = 1:
	If input buffer is empty, writes 0 to register
	C, else writes 1 to register C
A = 2:
	Clears input buffer
A = 3:
	If key written in register C is pressed, writes
	1 to register A, else writes 0 to register A.
